2. Identité UX et Design
Style visuel
 Design : Minimaliste, architectural, inspiré de l’ingénierie moderne
 Ambiance : Luxe discret (pas ostentatoire)
 Couleurs :
o Noir charbon / Gris anthracite
o Blanc cassé
o Accent cuivre ou doré industriel (très subtil) (voir le logo d’ITM)
 Typographies :
o Titres : Sans-serif moderne (ex: Montserrat / Poppins)
o Textes : Inter / Open Sans
UX principles
 Navigation simple (5–6 rubriques max)
 Grands visuels immersifs
 Peu de texte, mais des mots forts
 CTA sobres mais rassurants
 Site pensé desktop-first, puis mobile
